One Detailed Way (Recommended)

  1. Run complete suite with auto Slack: npm run test:notify
  2. If needed, run month only: npm run test:only -- "tests/timesheet by month" --project=chrome
  3. Open local report in browser: npx playwright show-report
  4. Slack gets summary automatically; if bot token + channel are set, HTML report is also uploaded.
  5. If Slack summary is needed without rerun: npm run report:slack

Tip: If a test is skipped, check its skip reason in test-results/results.json.
